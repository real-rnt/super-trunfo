### Projeto Super trunfo da faculdade
---
Neste projeto estou desenvolvendo um jogo de super trunfo puramente em C
na matéria fundamentos de programação em C da faculdade de Engenharia de Software.

É um programa bem simples mas que coloca em prova os fundamentos da programação.
Basicamente o usuário juntamente com um amigo ou sosinho, insere dados de cartas
que representam cidades, cada cidade tem seu atributo.
No fim os atributos são comparados para ver qual cidade se sobressai.
